# KAAT - Every Language, Every Family

![KAAT Logo](branding/logos/logo-main.png)

> *L'avenir de la lecture pour la jeunesse africaine.*  
> Free, multilingual educational content for Cameroonian children — preserving linguistic heritage through accessible technology.

**Mission:** A future where technology waters roots instead of erasing them.

---

## Quick Overview

KAAT is a free digital library offering multilingual books and learning materials for Cameroonian children. This project combines technology, culture, and education to address language extinction and limited access to quality educational resources in local languages.

- 10+ Cameroon languages supported
- Mobile-first & offline-capable — works on low-bandwidth devices
- 100% free for personal and educational use
- Submission to [IDEM Africa Challenge 2025](https://idem.africa/fr/challenge)

## Tech Stack

**Frontend:** React.js + Vite  
**Styling:** Pure CSS  
**Storage:** localStorage (bookmarks)  
**Features:** Service workers (offline), PDF export, responsive design  

---

## Key Features

- Browse & Search books by language, age group, and type
- Interactive Reader with bookmarking, text sizing, and page navigation
- Download as PDF or print for offline use
- Responsive Design optimized for all screen sizes and slow connections
- Offline Support via service worker caching
